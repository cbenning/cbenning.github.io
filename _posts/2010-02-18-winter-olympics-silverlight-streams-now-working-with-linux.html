--- layout: post title: Winter Olympics Silverlight streams now working with linux! categories: - Linux - Opensource tags: [] status: publish type: post published: true meta: _edit_last: '1' --- I finally got the streams on http://ctvolympics.ca working on linux using firefox 3.5.7 and moonlight 3 preview 2. Once the plugin is installed, performance may not be super hot. To remedy this, disable the effects in moonlight by running this command in a terminal: 
> MOONLIGHT_OVERRIDES=effects=no
Then restart firefox. For other issues, checkout the moonlight issues page.
