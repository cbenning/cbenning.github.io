--- layout: post title: How to Force Snow Leopard+ to consider a shell script safe. categories: [] tags: [] status: publish type: post published: true meta: _edit_last: '1' --- You can tell OSX Snow Leopard to consider shell scripts safe to execute without a user prompt. This is useful for automation of certain tasks. You can use any UTI mime-type to allow any filetype you wish. To do so, place the below text in ~/Library/Preferences/com.apple.DownloadAssessment.plist {% highlight html %} LSRiskCategorySafe LSRiskCategoryContentTypes public.shell-script {% endhighlight %}
